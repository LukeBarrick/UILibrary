API Reference

Property Type Default Description
type string 'info' Toast type: 'success', 'warning', 'error', 'info'
title string '' Toast title displayed in header
message string '' Main toast message content
duration number 5000 Auto-dismiss duration in milliseconds
position string 'top-right' Toast position on screen
closable boolean true Show close button
persistent boolean false Prevent auto-dismiss
