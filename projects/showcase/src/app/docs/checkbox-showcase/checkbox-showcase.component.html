Checkbox Component

The uilibrary-checkbox component provides accessible checkboxes with multiple variants, sizes, and states. It supports form integration and various styling options.

Checkbox Variants

Checkboxes come in three variants: branded, pass (success), and fail (error).

API Reference

Property Type Default Description
variant string 'branded' Checkbox style: 'branded', 'pass', 'fail'
size string 'small' Checkbox size: 'small', 'large'
checked boolean false Whether the checkbox is checked
disabled boolean false Whether the checkbox is disabled
hideLabel boolean false Hide the label visually (for accessibility)
labelPosition string 'right' Label position: 'left', 'right'
