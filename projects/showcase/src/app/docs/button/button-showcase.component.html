Button Component

The uilibrary-button component provides consistent, accessible buttons with multiple appearances, sizes, and states. It supports icons and can be disabled for form validation or loading states.

Button Sizes

Buttons are available in two sizes: normal (default) and small.

Normal Size (Default)

Normal Primary Normal Secondary Normal Success Normal Danger

Small Size

Small Primary Small Secondary Small Success Small Danger 
<!-- Normal size (default) -->

        <!-- Small size -->
      <uilibrary-button appearance="primary" size="small">Small Button</uilibrary-button>

API Reference

Property Type Default Description
appearance string 'primary' Button appearance: 'primary', 'secondary', 'primary-success', 'secondary-success', 'primary-delete', 'secondary-delete'
size string 'normal' Button size: 'normal', 'small'
disabled boolean false Whether the button is disabled
icon string undefined Icon name to display in the button
