Select

Overview

The Select2Component, projected with [ui-select], is a custom Angular select control designed to integrate seamlessly with the FormFieldComponent via the UIFormFieldControl<T> interface. It supports single and multi-select modes, searchability, templates, virtual scroll, and many additional customization options.

👷 Component Selector

<uilibrary2-select ui-select></uilibrary2-select>

Project this component into a <uilibrary2-form-field> wrapper to benefit from standard label, prefix, suffix, and validation integration. 

<uilibrary2-form-field>
  <label ui-label>Country</label>
  <uilibrary2-select ui-select
                     [items]="countries"
                     [(ngModel)]="selectedCountry"
                     [placeholder]="'Choose one...'">
  </uilibrary2-select>
</uilibrary2-form-field>

🧠 Integrated Interfaces

  • UIFormFieldControl<T> — exposes the control state to the FormFieldComponent.
  • ControlValueAccessor — enables usage with Angular Forms, FormControl, FormGroup.

📦 Inputs

Name Type Description
itemsany[]Array of selectable items.
valueanyCurrent selected value(s).
placeholderstringDisplayed when no selection is made.
multiplebooleanEnable multi-selection mode.
readonlybooleanDisables user interaction.
loadingbooleanIndicates loading state.
searchablebooleanEnable search/filter input.
virtualScrollbooleanOptimize rendering for long lists.
trackByFnFunctionCustom tracking function.
searchFnFunctionCustom filtering logic.
clearablebooleanWhether selection can be cleared.
notFoundTextstringMessage shown when no items match.
prefillFirstOptionbooleanPre-select first option on init.
closeOnSelectbooleanWhether to close dropdown after selection.

🔁 Form Control State Mapping

Property Description
shouldLabelFloattrue if selection exists or control is focused.

touchedtrue if control has been blurred.
dirtytrue if value has been modified.
emptytrue if no value selected.

🧩 Template Customization

Use Angular's ng-template with #labelTemplate and #optionTemplate inside the select to customize label and option rendering. 

<uilibrary2-select [items]="countries" [(ngModel)]="selected">
  <ng-template #labelTemplate let-selected>
    {{"{{ selected?.name || 'Select country' }}"}}
  </ng-template>

  <ng-template #optionTemplate let-item>
    🌍 {{"{{ item.name }}"}}
  </ng-template>
</uilibrary2-select>

💡 Notes

  • The select automatically integrates with FormFieldComponent via the UIFormFieldControl contract.
  • The FormFieldComponent can float labels, display validation state, and apply styles automatically based on this control’s state.
  • Set [ui-select] if you plan to project this select into a form field wrapper (optional if selector already matches).

📌 Example: With Floating Label and Validation

<form [formGroup]="form">
  <uilibrary2-form-field>
    <label ui-label>Region</label>
    <uilibrary2-select
      ui-select
      formControlName="region"
      [items]="regions"
      [placeholder]="'Select a region'"
      [searchable]="true">
    </uilibrary2-select>
  </uilibrary2-form-field>
</form>
