API Reference

Component Purpose Usage
uilibrary2-datepicker-input Single date selection component <uilibrary2-datepicker-input [(ngModel)]="date"></uilibrary2-datepicker-input>
uilibrary2-date-range-input Date range selection wrapper <uilibrary2-date-range-input><input startDate><input endDate></uilibrary2-date-range-input>
startDate directive Marks input as start date in range <input startDate formControlName="start"/>
endDate directive Marks input as end date in range <input endDate formControlName="end"/>
