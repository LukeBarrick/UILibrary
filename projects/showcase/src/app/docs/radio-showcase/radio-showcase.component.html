API Reference

Component Property Type Default Description
uilibrary-radio-button value any null Value of the radio button
checked boolean false Whether the radio button is checked
disabled boolean false Whether the radio button is disabled
(change) EventEmitter - Emitted when checked state changes
uilibrary-radio-group disabled boolean false Whether the entire group is disabled
(change) EventEmitter - Emitted when selection changes
