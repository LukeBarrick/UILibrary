API Reference

Property Type Default Description
items any[] [] Array of items to display in dropdown
multiple boolean false Enable multiple selection
clearable boolean false Show clear button to reset selection
placeholder string '' Placeholder text when no selection
bindLabel string 'label' Property name for display label
bindValue string null Property name for value binding
maxSelectedItems number null Maximum selectable items (multi-select)
closeOnSelect boolean true Close dropdown after selection
useCustomTemplate boolean true Use custom templates for display
prefillFirstOption boolean false Automatically select first option
searchFn function null Custom search function
