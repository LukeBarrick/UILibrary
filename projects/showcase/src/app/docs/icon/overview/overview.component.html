Icons

Overview

The Icon2Component is a lightweight Angular component for rendering SVG icons from a centralized icon registry. It supports configurable sizes and themed appearances via input properties.

This component uses the <svg-icon> component under the hood to load icons by name. The list of available icon names will be published soon. Icon list documentation: TBD

👷 Component Selector

<uilibrary2-icon></uilibrary2-icon> 

<uilibrary2-icon 
  [name]="'checkmark'" 
  [size]="'medium'" 
  [appearance]="'primary'">
</uilibrary2-icon>

📦 Inputs

Input Type Description
name string Required. The identifier of the icon to load from the shared icon set.
size 'small' | 'medium' | 'large' Optional. Sets the icon’s width and height (in pixels).
appearance 'primary' | 'secondary' | 'tertiary' | 'light-gray' | 'dark-gray' | 'custom' Optional. Determines the icon’s fill color.

🎨 Sizes

Size Width × Height (px)
'small'30 × 30
'medium' (default)40 × 40
'large'80 × 80

🌈 Appearances

Appearance Fill Color
'primary'var(--primary)
'secondary'var(--secondary)
'tertiary'var(--tertiary)
'light-gray'var(--light-gray)
'dark-gray'var(--dark-gray)
'custom'currentColor
defaultvar(--secondary)

Note: Appearance colors use CSS variables to align with your design system. Customize var(--*) in global stylesheets.

💡 Notes

  • This component is best used with an icon registry and prebuilt SVG icon set.
  • It delegates rendering to the <svg-icon> component and applies style via property bindings.
  • Size and color styles are applied using inline styles via [svgStyle].
