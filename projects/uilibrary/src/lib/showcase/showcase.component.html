Limitations on Content Projection

Angular's content projection runs at compile time. This is the mechanism used that allows us to project the extra menu content into the various menu's. As such it has the following limitations:

  • Static projection slots (everything is done at compile time)
  • Only one projection per slot. rightAlignedContent and extraMobileMenuContent cannot be used on the same element
  • Project content does nto play well with conditional logic @for/@if etc

See https://brandonclapp.com/how-to-use-ng-content-for-content-projection-in-angular-a-basic-overview for a good write up on how ng-content works and its limitations.
