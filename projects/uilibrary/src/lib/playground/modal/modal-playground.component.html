Sidebar Modal Routing

Routes flagged with outlet: 'sidebar' render into a hidden <router-outlet name="sidebar">. A headless <ui-sidebar-modal-outlet> in the shell template watches NavigationEnd events and bridges them to UiModalService — opening a slide-in CDK overlay drawer. Dismissing the drawer (backdrop, Escape, or the Close button) clears the outlet from the URL, keeping router state and modal state in sync.

Drawer demos

Watch the browser URL bar — it changes to /playground/(modal//sidebar:sidebar-profile) when the drawer opens and returns to /playground/modal when it closes.
