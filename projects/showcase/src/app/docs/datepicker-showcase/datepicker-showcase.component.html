API Reference

Components

Component Purpose Usage
uilibrary-datepicker-input Single date selection component <uilibrary-datepicker-input [(ngModel)]="date"></uilibrary-datepicker-input>
uilibrary-date-range-input Date range selection wrapper <uilibrary-date-range-input><input startDate><input endDate></uilibrary-date-range-input>

Directives

Directive Purpose Usage
startDate Marks input as start date in range <input startDate formControlName="start"/>
endDate Marks input as end date in range <input endDate formControlName="end"/>
ui-prefix Add prefix content (icons, text, etc.) <uilibrary-icon ui-prefix name="icon-air"></uilibrary-icon>
ui-suffix Add suffix content (icons, text, etc.) <uilibrary-icon ui-suffix name="icon-air"></uilibrary-icon>

Input Properties

Property Type Default Description
closeOnSelection boolean true Whether to close the calendar after selecting a date/range
editable boolean true Whether the input field is editable by typing
disabled boolean false Disables the date picker input

Methods

Method Parameters Returns Description
open() none void Programmatically opens the calendar dropdown
