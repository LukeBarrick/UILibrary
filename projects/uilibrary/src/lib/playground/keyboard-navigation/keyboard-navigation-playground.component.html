Keyboard Navigation

Interactive Grid

Use the arrow keys (↑ ↓ ← →) to move focus between cells. The [uiCellNav] directive registers each cell with the NavRegistryService so keyboard navigation is handled automatically.

@for(row of gridRows; track row; let rowIndex = $index) { @for(cell of row; track cell; let colIndex = $index) {
{{ cell }}
} }
Note: Apply [uiCellNav] to any focusable container element. Provide [rowIndex] and [colIndex] to define its position in the grid. The directive automatically sets tabIndex="0" and role="gridcell".
