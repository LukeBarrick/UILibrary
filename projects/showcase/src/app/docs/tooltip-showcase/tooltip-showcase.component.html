Tooltip Component

The uilibrary-tooltip component provides contextual information on hover or focus. It supports multiple placements and can be used with any UI element.

Basic Tooltip Placements

Tooltips can be positioned in four main directions around the target element.

Top Tooltip This tooltip appears above the element.
Right Tooltip This tooltip appears to the right of the element.
Bottom Tooltip This tooltip appears below the element.
Left Tooltip This tooltip appears to the left of the element.

Hover over the tooltip buttons above to see different placements

<uilibrary-tooltip placement="top">
        <span title>Top Tooltip</span>
        <span content>This tooltip appears above the element.</span>
        </uilibrary-tooltip>

        <uilibrary-tooltip placement="right">
        <span title>Right Tooltip</span>
        <span content>This tooltip appears to the right.</span>
      </uilibrary-tooltip>

Accessibility and Best Practices

Tooltips should enhance accessibility and provide valuable context without overwhelming users.

Accessibility Guidelines

  • Keep content concise: Tooltips should provide brief, helpful information
  • Use appropriate titles: Title should summarize the tooltip purpose
  • Consider keyboard users: Tooltips should be accessible via keyboard navigation
  • Don't rely solely on tooltips: Critical information should be visible without hovering

Example: Good vs Poor Tooltip Usage

✓ Good Example
Save Document Save your changes to the current document. Shortcut: Ctrl+S
✗ Poor Example
Button This button does something when you click it and it might be important but we're not sure what exactly it does or why you would want to use it.

API Reference

Property Type Default Description
placement string 'top' Tooltip position: 'top', 'bottom', 'left', 'right'
margin string '' Additional margin: 'left', 'right', 'top', 'bottom'

Content Slots

Slot Purpose Usage
title Tooltip title/header <span title>Tooltip Title</span>
content Tooltip body content <span content>Tooltip description</span>
