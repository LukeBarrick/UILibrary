Status Tag Component

The uilibrary-status-tag component provides visual status indicators with different styles and colors to communicate state, priority, or category information.

Status Types

Status tags are available in multiple predefined types to represent different states and meanings.

Success Success Status
Warning Warning Status
Error Error Status
Info Info Status
Primary Primary Status
Secondary Secondary Status
Inactive Inactive Status

API Reference

Property Type Default Description

Available Status Types

Status Use Case Visual Style Example
success Completed, approved, active states Green background with white text Active
warning Attention required, pending review Yellow/orange background Pending
error Failed, rejected, critical issues Red background with white text Failed
info Informational, in progress Blue background with white text Processing
primary Default, important items Primary brand color Default
secondary Secondary information Gray background Draft
inactive Disabled, offline, archived Muted colors Archived
pending Awaiting action or approval Light blue/gray background Waiting
