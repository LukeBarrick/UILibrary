Navigation Component

The uilibrary-navigation component provides a responsive navigation bar with support for logos, links, dropdowns, and mobile-friendly collapsible menus. It supports content projection for custom elements.

Navigation with Dropdown Menus

Navigation supports nested dropdown menus. Use the hideNLabelOnNestedChildren option for mobile menus.

Should there be nested dropdowns, say within the mobile menu, it is possible to hide the label of the parent item which would have triggered the dropdown with hideNLabelOnNestedChildren

Different Link Types

Navigation supports different types of links: routes, external URLs, and target blank URLs.

Limitations on Content Projection

Angular's content projection runs at compile time. This mechanism allows projecting extra menu content into various menus, but has limitations:

  • Static projection slots (everything is done at compile time)
  • Only one projection per slot. rightAlignedContent and extraMobileMenuContent cannot be used on the same element
  • Projected content does not play well with conditional logic *ngIf etc

See this article for a good write up on how ng-content works and its limitations.

API Reference

Property Type Default Description
links NavigationLink[] [] Array of navigation links to display
showRightAlignedContent boolean false Whether to show right-aligned content area
hideRightAllignedContentOnMobile boolean false Hide right-aligned content on mobile devices
showExtraMobileMenuContent boolean false Whether to show extra content in mobile menu
